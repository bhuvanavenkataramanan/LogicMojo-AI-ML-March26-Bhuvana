# How to run this project

The data set for the project is in the folder python_project_aiml_logicmojo_dataset.

Modules needed for running the project
* pandas
* matplotlib.pyplot
* seaborn
* numpy

The sequence to follow
* Load the datasets from the csv format in the files to pandas dataframes
* Clean up the data and enhance it
* Merge the tables
* Run the EDA analysis over the merged/normalized table (orders_master)
* Plot the graphs for analysis
